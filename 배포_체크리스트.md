# ✅ 배포 체크리스트

이 체크리스트를 따라 앱 배포 준비를 완료하세요.

## 🔧 환경 설정

### 필수 도구 설치

- [ ] Node.js 18 이상 설치
- [ ] Java JDK 11 이상 설치
- [ ] Android Studio 설치
- [ ] Android SDK 설치 (API Level 21 이상)

### 환경 변수 설정

- [ ] ANDROID_HOME 환경 변수 설정
- [ ] JAVA_HOME 환경 변수 설정
- [ ] PATH에 Android 도구 추가

**설정 확인:**
```powershell
$env:ANDROID_HOME
$env:JAVA_HOME
java -version
adb version
```

## 📱 Android 프로젝트

### 프로젝트 초기화

- [ ] Android 프로젝트 초기화 완료
  ```powershell
  .\scripts\init-android.ps1
  ```

### 프로젝트 설정

- [ ] 앱 이름 설정 (`android/app/src/main/res/values/strings.xml`)
- [ ] 패키지 이름 설정 (`android/app/build.gradle`)
- [ ] 버전 번호 설정 (versionCode, versionName)
- [ ] 최소 SDK 버전 설정 (minSdkVersion 21 이상)

### 권한 설정

- [ ] 카메라 권한 추가
- [ ] 인터넷 권한 추가
- [ ] 저장소 권한 추가
- [ ] 알림 권한 추가 (Android 13+)

### 아이콘 설정

- [ ] 앱 아이콘 생성 및 설정
- [ ] 모든 해상도 아이콘 준비 (mdpi ~ xxxhdpi)
- [ ] Adaptive Icon 설정 (선택사항)

## 🔐 서명 설정 (Release 빌드)

### 서명 키 생성

- [ ] 서명 키 생성 완료

## 🏗️ 빌드

### 빌드 테스트

- [ ] Debug 빌드 성공
  ```powershell
  .\scripts\build-android.ps1 -BuildType debug
  ```

- [ ] Debug APK 설치 및 실행 테스트
- [ ] 주요 기능 동작 확인
- [ ] 크래시 없는지 확인

### Release 빌드

- [ ] Release 빌드 성공
  ```powershell
  .\scripts\build-android.ps1 -BuildType release
  ```

- [ ] AAB 빌드 성공
  ```powershell
  cd android
  .\gradlew.bat bundleRelease
  ```
