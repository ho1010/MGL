## 🚀 복사 후 다음 단계

### 1단계: 의존성 설치
```powershell
cd figma-mcp-integration
bun install
```

### 2단계: 프로젝트 빌드
```powershell
bun run build
```

### 3단계: Cursor MCP 설정
`~/.cursor/mcp.json` 파일 생성/편집:
```json
{
  "mcpServers": {
    "TalkToFigma": {
      "command": "bun",
      "args": ["C:/Users/Seol/Documents/GitHub/MGL/figma-mcp-integration/src/talk_to_figma_mcp/server.ts"]
    }
  }
}
```

### 4단계: WebSocket 서버 실행
```powershell
cd figma-mcp-integration
bun socket
```

### 5단계: Figma 플러그인 설치
1. Figma 실행
2. Plugins > Development > New Plugin
3. Link existing plugin 선택
4. `manifest.json` 파일 선택

### 6단계: 연결 및 사용
1. Figma 플러그인에서 Connect 클릭
2. Cursor에서 `join_channel` 사용
3. Cursor AI로 Figma 제어 시작!
